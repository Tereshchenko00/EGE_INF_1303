### Кодирование текстовой информации

**I = n * i**

где:

- **n** — количество символов
- **i** — количество бит на 1 символ (кодировка)


### Кодирование графической информации

Рассмотрим некоторые понятия и формулы, необходимые для решения ЕГЭ по информатике данной темы.

- **Пиксель** – это наименьший элемент растрового изображения, который имеет определенный цвет.
- **Разрешение** – это количество пикселей на дюйм размера изображения.
- **Глубина цвета** — это количество битов, необходимое для кодирования цвета пикселя.
- Если глубина кодирования составляет **i** битов на пиксель, код каждого пикселя выбирается из _2i_ возможных вариантов, поэтому можно использовать не более _2i_ различных цветов.
- 
![alt text](<IMAGES/Pasted image 20250525155953.png>)
#### Формула для нахождения количества цветов в используемой палитре:

i = log2N

- **N** — количество цветов
- **i** — глубина цвета

![!\[\[Pasted image 20250525160019.png\]\]](<IMAGES/Pasted image 20250525160019.png>)
Или получаем:

N = 2<sup>i</sup>

- **В цветовой модели RGB** (красный (R), зеленый (G), синий (B)): R (0..255) G (0..255) B (0..255) -> получаем _28_ вариантов на каждый из трех цветов.
- R G B: 24 бита = 3 байта — **режим True Color** (истинный цвет)

#### Найдем **формулу объема памяти для хранения растрового изображения**:

I = M * N * i

где:

- **I** — объем памяти, требуемый для хранения изображения
- **M** — ширина изображения в пикселях
- **N** — высота изображения в пикселях
- **i** — глубина кодирования цвета или разрешение

#### Или, можно формулу записать так:

I = M * i битов

- где **M** – количество пикселей (x * y) и **i** – глубина кодирования цвета (разрядность кодирования)

* для указания объема выделенной памяти встречаются разные обозначения (_V_ или _I_).

- Следует также помнить формулы преобразования:

1 Мбайт = 2<sup>20</sup> байт = 2<sup>23</sup> бит,  
1 Кбайт = 2<sup>10</sup> байт = 2<sup>13</sup> бит

### Кодирование звуковой информации

Познакомимся с понятиями и формулами, необходимыми для решения заданий 7 ЕГЭ по информатике.

- **Оцифровка или дискретизация** – это преобразование аналогового сигнала в цифровой код.

[![](https://labs-org.ru/wp-content/uploads/1-19.png)](https://labs-org.ru/wp-content/uploads/1-19.png)
<center>Дискретизация, объяснение задания 7 ЕГЭ</center>

- **T** – интервал дискретизации (измеряется в с)
- **ƒ** — частота дискретизации (измеряется в Гц, кГц)

- **Частота дискретизации** определяет количество отсчетов, т.е. отдельных значений сигнала, запоминаемых за 1 секунду. Измеряется в герцах, **1 Гц** (один герц) – это один отсчет в секунду, а, например, **7 кГц** – это **7000** отсчетов в секунду.
- **Разрядность кодирования (глубина, разрешение)** — это число битов, используемое для хранения одного отсчёта.

[![](https://labs-org.ru/wp-content/uploads/2-7.png)](https://labs-org.ru/wp-content/uploads/2-7.png)<center>Разрядность кодирования</center>

  
- Получим **формулу объема звукового файла:**

Для хранения информации о звуке длительностью _t_ секунд, закодированном с частотой дискретизации _ƒ_ Гц и глубиной кодирования _β_ бит требуется бит памяти:

Её легко запомнить. **<font color="#00b0f0">Объём записанного файла равен произведению всех остальных параметров</font>**. Важно соблюдать единицы измерения.

![!\[\[Pasted image 20250525160132.png\]\]](<IMAGES/Pasted image 20250525160132.png>)

I = β * ƒ * t * S

- **I** — объем
- **β** — глубина кодирования
- **ƒ** — частота дискретизации
- **t** — время
- **S** — количество каналов

_S_ для моно _= 1_, для стерео _= 2_, для квадро _= 4_
