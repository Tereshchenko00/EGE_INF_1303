## Проверка числа на простоту 

```python
def isSimple(n): #название поставь какое удобно
    return n > 1 and all(n % i for i in range(2, int(n**0.5) + 1))
```

- проверка `n > 1` - простые числа всегда больше 1
- далее генератор, который генерирует остатки деления числа `n` на `i` в промежутке от `2` до `int(n**0.5)+1` - корня из `n`
- функция `all()` возвращает `True` только когда все аргументы(в нашем случае остатки) ненулевые (эквивалентны `True`)
