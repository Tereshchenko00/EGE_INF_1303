## Суть задания
> У исполнителя есть две команды:
> 	1. прибавить 1
> 	2. умножить на 2
> Сколько существует программ


![[Pasted image 20250322123358.png]]
## Пример
```python
n = 10 #в данном примере
a = [0]*(n+1)
a[1] = 1

for i in range(2,n+1):
	a[i] += a[i-1]
	if i%2 == 0:
		a[i] += a[i//2]
print(a)


```
